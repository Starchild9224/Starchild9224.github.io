Nutrition E-book UX Design

Critique and Innovation of ProQuest Ebook Central

Overview

This project aims to develop an innovative e-textbook for a general university-level nutrition course. Unlike traditional e-textbooks and the existing ProQuest Ebook platform, this version will incorporate highly interactive features designed to enhance the learning experience. Throughout the development process, rigorous usability engineering principles and systematic user testing were applied to comprehensively optimize the system—resulting in a platform that not only innovates upon but also surpasses existing e-book solutions.
